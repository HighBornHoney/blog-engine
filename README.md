# Blog Engine

Простой блог, реализованный на чистом PHP (без фреймворков), с использованием MySQL и Smarty.

## Функционал

- Категории со статьями
- Список статей:
    - Сортировка (по дате / просмотрам)
    - Пагинация
- Страница статьи:
    - Счётчик просмотров
    - Похожие статьи
- Чистая MVC-архитектура
- Шаблонизатор Smarty
- Сидер для генерации тестовых данных

---

## Технологии

- PHP 8.1+
- MySQL
- Smarty
- Bootstrap 5

---

## Установка

### 1. Клонировать репозиторий

`git clone https://github.com/HighBornHoney/blog-engine.git`

`cd blog-engine`

### 2. Создать файл настроек проекта

`cp .env.example .env`

Изменить порт приложения и пароль от БД, если требуется

---

### 3. Запуск проекта

`docker compose up -d`

Создание БД, импорт sql дампа и установка composer зависимостей произойдут автоматически

Открыть в браузере:

http://localhost:9100

---

### 4. Заполнить базу (сидинг)

`docker compose exec php php database/seeders/Seeder.php`

---


## Архитектура

Проект построен по паттерну MVC:

- Controllers — обрабатывают запросы
- Models — работа с базой данных
- Views (Smarty) — отображение
- Core — роутер, база, базовые классы
